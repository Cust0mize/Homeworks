Стратегия:
1) Генерация примеров в математической игре, где для в зависимости от знака будет применяться необходимый алгоритм, к примеру для + мы будем брать любые числа из диапозона, для вычитания мы будем делать дополнительную проверку на то, что-бы число не было отрицательным, а для деления выбирать только числа которые будут делиться без остатка.
2) Загрузка сохранений, где мы будем в одном случае грузить сохранения из бинарника, а в дргугом из облака

Шаблонный метод:
1) Базовый тип врага который может атаковать игрока и несколько реализаций этого класса каждый из которых атакует его по разному
2) Базовы класс который может получать урон и несколько классов наследников которые получают этот урон по разному
